# 四釋 的數位映射

## 1. 定義 (Definition)
「四釋」是智者大師在《法華文句》中創立的語義解析方法。它要求對經文中的每一個關鍵字詞或概念，都從四個不同的維度進行深度解讀，以窮盡其內涵。

## 2. 四大解析層次 (The Four Layers)

### 2.1 因緣釋 (Causal Interpretation) —— 環境語境層
*   **功能**：分析該說法產生的時空背景、受眾需求與觸發事件。
*   **數位映射**：**Contextual Analysis / Event Trigger**。識別數據產生的環境參數與觸發條件。

### 2.2 約教釋 (Doctrinal Interpretation) —— 系統標準層
*   **功能**：根據天台宗的判教體系（五時八教），確定該說法在系統架構中的邏輯等級。
*   **數位映射**：**Standard Mapping / Hierarchy Classification**。將數據對齊到系統的版本控制與權限等級中。

### 2.3 [[本跡|本跡]]釋 (Origin & Trace Interpretation) —— 本體實例層
*   **功能**：區分該說法是屬於永恆的本體（本），還是適應當下的表現形式（跡）。
*   **數位映射**：**Ontological Mapping (Class vs. Instance)**。區分底層核心邏輯（Source Code）與前端執行實例（UI/API Call）。

### 2.4 觀心釋 (Contemplative Interpretation) —— 內核應用層
*   **功能**：將該說法直接映射回修持者的心性運作中，化理論為實踐。
*   **數位映射**：**Inner Loop Application / Self-Reflexive Logic**。將解析結果直接應用於系統的自我優化與反饋循環中。
