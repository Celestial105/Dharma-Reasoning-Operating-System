# 五重玄義 的數位映射

## 1. 定義 (Definition)
「五重玄義」是天台宗智者大師創立的一套標準化文獻（數據）審計協議。在深入讀取任何一部經典或數據塊之前，必須先從五個維度進行宏觀掃描，以確保對目標的全面理解。

## 2. 五大維度 (The Five Dimensions)

### 2.1 釋名 (Name / Semantics)
*   **定義**：解釋標題與術語的深層語義。
*   **數位映射**：**Lexical Analysis (詞法分析)**。定義數據的命名空間（Namespace）與核心標籤。

### 2.2 辨體 (Essence / Ontology)
*   **定義**：辨明該數據所依據的本體（通常是實相）。
*   **數位映射**：**Data Model (數據模型)**。確定數據的底層架構與存在性質（例如：它是性境還是獨影境）。

### 2.3 明宗 (Theme / Goal)
*   **定義**：明確該經典修行的核心宗旨與優化方向。
*   **數位映射**：**Objective Function (目標函數)**。系統運算的核心目的（如：收斂於「[[空|空]]」還是「[[佛性|佛性]]」）。

### 2.4 力用 (Function / Utility)
*   **定義**：該數據對用戶（眾生）產生的實際功用與解決的痛點。
*   **數位映射**：**Use Case (應用場景)**。該算法能解決什麼問題（如：破除執著、提升運算速度）。

### 2.5 判教 (Classification / Versioning)
*   **定義**：該數據在整個系統中的位置與等級。
*   **數位映射**：**System Hierarchy / Versioning**。該組件屬於系統的哪個模組，其權重與優先級為何。
