# 體相用 的數位映射

## 1. 定義 (Definition)
「體相用」是《[[大乘]]起信論》描述真如[[大乘]]的三個維度，後來成為東方哲學中分析事物的標準框架。
*   **體 (Essence)**：事物的本體、本質、不變的基石。
*   **相 (Attribute)**：事物的表徵、屬性、德相、可觀測的特徵。
*   **用 (Function)**：事物的功用、作用、能量的釋放、產生的影響。

## 2. 數位映射 (Digital Mapping) —— 標準對象模型 (Standard Object Model)
在數位佛堂中，每一個「原子積木（節點）」都可以用此模型來定義：
*   **體 (Class Definition)**：該節點在 Wiki 中的唯一 ID、類型與底層定義。
*   **相 (Metadata/Properties)**：該節點的標籤（Tags）、標題（Title）與關聯關係（Connections）。
*   **用 (Action/Method)**：該節點能為系統提供的邏輯價值（如：解決了什麼問題、提供了什麼算法）。
