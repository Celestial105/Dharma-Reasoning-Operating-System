# 四分 的數位映射

## 1. 定義 (Definition)
「四分」是唯識學中關於「[[識|識]]」之結構的四種成分。它完整描述了一個認知行為（或數據處理行為）從輸入到最終審計的全過程。

## 2. 四大組成部分 (The Four Components)

### 2.1 相分 (Object Part) —— 數據荷載層 (Payload)
*   **功能**：識所呈現的對象影子。
*   **數位映射**：**Input Data / Resource / Payload**。這是被處理的原始數據對象。

### 2.2 見分 (Subject Part) —— 執行觀測層 (Observer)
*   **功能**：識的觀測、認識功能。
*   **數位映射**：**Processor / Sensor / Observer Function**。負責對數據進行解析與運算的主體函數。

### 2.3 自證分 (Self-witnessing Part) —— 系統監控層 (Monitoring)
*   **功能**：對「見分」的認識結果進行自我證實與記錄。
*   **數位映射**：**System Logger / Performance Monitor**。這是系統內部的「監控器」，負責確認執行層（見分）是否正確完成了任務。

### 2.4 證自證分 (Evidence of self-witnessing Part) —— 安全審計層 (Audit)
*   **功能**：對「自證分」進行再驗證。自證分與證自證分互為見、相，形成閉環。
*   **數位映射**：**Security Auditor / Integrity Verifier**。這是「監控的監控」，負責審計監控日誌的真實性與完整性，確保系統沒有發生底層邏輯坍塌。
