# 無住生心 的數位映射

## 1. 定義 (Definition)
「無住生心」是《金剛經》修行的核心指引，也是六祖惠能大師[[開悟]]的關鍵。它要求系統在處理數據、產生輸出（生心）的同時，不被任何過往的狀態、預設的立場或固定的緩存所束縛（無住）。

## 2. 數位映射 (Digital Mapping)
*   **無狀態運算 (Stateless Computing)**：系統處理每一個 Request 時，都不依賴於 Session 的緩存執著。每一次「生心（輸出）」都是基於當下最純粹的數據輸入。
*   **動態上下文 (Dynamic Context)**：AI 不被過往的「偏見」所左右，實現真正的 **「零偏置輸出」**。
*   **實時顯化**：數據一過即消，不留殘影。
